# Frontend - Youth Talent Fund

## Yêu cầu hệ thống
- Node.js >= 16
- npm >= 8 (hoặc yarn/pnpm)

## Cài đặt
1. Mở terminal và chuyển vào thư mục `Frontend`:
   ```sh
   cd Frontend
   ```
2. Cài đặt các package:
   ```sh
   npm install
   ```

## Chạy dự án ở chế độ phát triển
```sh
npm run dev


Sau đó truy cập địa chỉ hiển thị trên terminal (thường là http://localhost:5173 hoặc http://localhost:3000).
